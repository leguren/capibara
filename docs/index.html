Autenticación

Todos los endpoints bajo /api/geo/1/ excepto /catalog* requieren un token de API. Enviá el header Authorization: Bearer cpb_tutoken en cada request.

Generá un token desde tus keys — se muestra una sola vez al crearlo.

Endpoints
GET /api/geo/1/query?lat=&lon= requiere API key
Consulta el punto (lat, lon) contra todas las fuentes activas. Parámetros opcionales: precision (decimales de redondeo, default 3) y domain (filtrar por dominio/s separados por coma).
GET /api/geo/1/catalog público
Catálogo de metadatos: versión activa, dominios disponibles y conteos.
GET /api/geo/1/catalog/domains público
Lista de dominios temáticos con cantidad de capas y fuentes por dominio.
GET /api/geo/1/catalog/layers?domain= público
Lista de capas publicadas, opcionalmente filtradas por dominio/s.
GET /api/geo/1/catalog/coverage?lat=&lon= público
Qué capas y dominios cubren un punto, sin ejecutar las consultas — útil antes de gastar tu cuota.
