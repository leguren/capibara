¿Cómo funciona?

La API pública no lee directamente de la base de datos de fuentes — lee el último snapshot publicado.

Esto significa que podés agregar y modificar fuentes sin afectar a los clientes hasta que hagas una publicación.

Cada publicación tiene un version_label semántico (v1.0.0, v1.0.1…) y queda registrado con quién la hizo y cuándo.

Seguridad
Las credenciales de conexión (tokens, contraseñas) se eliminan automáticamente del config antes de guardarlo. El snapshot es seguro para auditar.
