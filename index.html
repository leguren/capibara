Datos geoespaciales sobre cualquier coordenada

Una sola API query → múltiples fuentes → respuesta unificada. Capibara consulta un punto geográfico y devuelve todo lo disponible de fuentes dispersas, en un solo llamado.

Así se ve una consulta
curl "https://capibara-ten.vercel.app/api/geo/1/query?lat=-34.603&lon=-58.382" \
  -H "Authorization: Bearer cpb_tutoken"
Múltiples fuentes, una respuesta
Capibara unifica fuentes WFS, ArcGIS, CSV, GeoJSON y JSON detrás de un único endpoint.
Sin almacenamiento propio
Nunca guarda datos de las fuentes — consulta en tiempo real con caché progresiva según la frecuencia de cada capa.
Organizado por dominios
Geografía, catastro, riesgo hídrico, transporte y más — filtrá por el dominio que te interesa.
